leetcode-665

2020.06.03 lxlandzyq  热度 ℃

题目

给你一个长度为 n 的整数数组，请你判断在 最多 改变 1 个元素的情况下，该数组能否变成一个非递减数列。

我们是这样定义一个非递减数列的： 对于数组中所有的 i (0 <= i <= n-2)，总满足 nums[i] <= nums[i + 1]。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/non-decreasing-array

分析

先分析题可能出现的情况
凸形状： 2，3，6，4 （其中6为凸起现状）
凹形状： 3，4，1，4 （其中1为需要修改的数值
判断凹或凸最多出现一次，即为true，否则为false
如何判断凹或凸需要三个值
当前值 index
index的前一个值 pre-index
pre-index的前一个值 p-pre-index
若index>pre-index 则continue
若index<pre-index
【若p-pre-index>index 即前两个值都大于index 则为凹需要修改index为pre-index 即修改一次 】
【若p-pre-index<index 即前-前一个值小于index，则表明是pre-index的值高出了，则将pre-index修改为index，修改一次 为凸】
