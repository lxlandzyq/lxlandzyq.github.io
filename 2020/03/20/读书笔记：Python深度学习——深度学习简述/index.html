读书笔记：Python深度学习—-深度学习简述

2020.03.20 lxlandzyq  热度 ℃

前述

主要通过自己的笔记总结，和书上的知识来对概念的理解。此文章主要对深度学习的大致过程进行梳理与总结。

深度学习

机器学习是将输入的数据（比如图像）——>目标(标签为猫)，通过映射来将图片与它标签内容进行一一映射。此过程则是通过观察许多的输入和目标的实例来完成的。
深度学习即是通过一些列简单的数据变换（即为层的作用），即为不断对输入的数据对其指定的路线的权值比重增大。通过此来实现输入到目标的映射

深度学习过程

权值的作用

每一层在输入的数据上找到其权重，即为最符合其对应标签，则其权重则越大。在每一层分配好其适应的权重的时候。最后一层中，输出值越大即为其符合的标签。


神经网络是由其权重来参数化

如何找到这些权重

对于这些权值，不是凭空捏造的，需要进行不断训练才可以得到其每一层对应的参数的权值。如何找到这些权值，就是在训练中通过神经网络的损失函数来衡量当前输出值与预期值之间的距离。

损失函数用来衡量网络输出结果的质量
通过上图可以知道，就是在通过损失还是来计算出预测值与真实目标值之间的距离

通过损失值来分配权值

有了损失值，则对每一层神经元之间的权重就要进行调节，调节到使其损失值最小。通过距离最为反馈信号对每层权重进行微调，用以降低当前对应的损失值，此调节有优化器完成。即为方向传播。

将损失值作为反馈信号来调节权重

一开始的权重是随机赋值的，不断进行权重的微调，损失值降低将这些数值循环重复足够多次数，得到权重值可以使损失函数最小。使其输出目标值尽可能的接近目标值。
