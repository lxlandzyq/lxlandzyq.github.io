github 换成token登录（随口一说）

2021.12.30 lxlandzyq  热度 ℃

随口一说

今天想记录一些东西，最后发现hexo一直不能将我的博客push上去，我知道我的git clone一直在学校网用不了，不知道为啥(报错443)，一般都用手机上的网进行git clone。我以为不能将博客push上去是这个原因，因此今天就找到了解决的办法：
参考链接：https://juejin.cn/post/6844904193170341896
以后还是这个问题就可以找它。用作记录保存。

如果以后还报443这个错误，根据上面的文档多刷新几次。

但是后来发现还是不能将其文章push上去

报错

注意在密码处，输入密码处，我的密码我是百分之百正确的，经过上网冲浪知道，其github不使用密码登录了，使用token，就要去github上申请一个token。再进行hexo d的时候密码处粘贴的为token。
将其存入钥匙串里面：参考链接:https://iphysresearch.github.io/blog/post/programing/2021-08-13-token-authentication-requirements-for-git-operations/

参考链接:https://blog.csdn.net/weixin_41010198/article/details/119698015
