将本地文件的代码git到github上面的流程，远程服务。
第一：git clone github上面的仓库地址
第二：修改相应的文件
第三：git add 修改过的文件// 如git add index.html
第四：git commit -m '注释，说明修改的内容是说明'
第五：git push 提交到github上面
二三四步可以根据自己情况修改，最后在git push
